Code2Art

코드를 예술 작품으로 변환하는 프로젝트

GitHub Repository

Summary

당신의 코드를 예술로 승화시키세요. Code2Art는 코드의 구조와 로직을 분석하여 거장들의 화풍으로 재해석된 고화질 추상화를 생성해주는 혁신적인 서비스입니다. 로컬 분석을 통해 데이터 유출 걱정 없이 나만의 마스터피스를 탄생시킬 수 있습니다.

Key Features

  • 지능형 코드 분석: 브라우저 내 로컬 AST 파싱을 통해 8대 예술 지표 추출
  • 거장들의 화풍: 칸딘스키, 반 고흐, 모네, 달리, 바스키아 등 화풍 선택/추천
  • AI 도슨트: 생성된 작품이 코드의 어떤 기술적 특징을 반영했는지 예술 비평가의 시선으로 해설
  • 고화질 렌더링: DALL-E 3 및 Gemini 등을 연동한 미술관 퀄리티의 이미지 생성

Information

미니멀한 화이트/그레이 테마는 코드의 정밀함과 예술적 창의성을 상징합니다.
