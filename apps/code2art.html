Code2Art

코드를 예술 작품으로 변환하는 프로젝트

🐙 GitHub Repository

Summary

당신의 코드를 예술로 승화시키세요. Code2Art는 단순한 코드 시각화를 넘어, 작성된 코드의 구조적 특징과 로직의 아름다움을 정밀하게 분석하여 위대한 거장들의 화풍으로 재해석된 고화질 추상화를 생성하는 혁신적인 웹 브라우저 예술 생성기입니다. 모든 분석은 로컬 환경에서 안전하게 이루어지며, 미니멀한 화이트/그레이 테마가 오롯이 당신의 코드 작품에 집중하게 해줍니다.

Key Features

  • 최첨단 지능형 코드 분석: 브라우저 내장 로컬 AST 파싱 기술을 통해 복잡도, 모듈성 등 8대 예술적 지표를 안전하게 추출
  • 거장의 터치: 칸딘스키, 반 고흐, 클로드 모네, 살바도르 달리 등 예술 사조별 화풍 선택 및 AI 추천 기능
  • 생성형 AI 도슨트: 완성된 작품이 코드의 어떤 기술적 특성을 반영해 탄생했는지 미술 평론가의 시선으로 깊이 있게 해설
  • 박물관 수준의 렌더링: 최신 DALL-E 3 및 Gemini Vision 모델 등을 연동하여 초고화질(4K) 아트워크 생성 및 다운로드 지원
